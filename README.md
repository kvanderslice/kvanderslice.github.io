# Portfolio Site

#### Getting started: Static site
* `$ cd app`
* `$ sudo gem install sass`
* `$ npm install`
* `$ gulp`


#### Getting started: WP theme
* Install WP locally: `$ cd <project>; wp core download`
* Add dev db to Sequel Pro


Compiling styles: WP theme
* `$ cd <theme>`
* `$ sudo gem install sass`
* `$ npm install`
* `$ gulp`
